## Quick Facts

- **Dev**: `npm run dev` → http://localhost:5173
- **Seed DB**: `npm run db:seed` (first run; start.bat does this automatically)
- **DB file**: `data.db` at project root (WAL mode)
- **Languages covered**: Python, JavaScript, TypeScript, Go, Rust, Kotlin, Swift, C#
- **i18n**: EN + PT-BR, `src/lib/i18n/translations.ts`
